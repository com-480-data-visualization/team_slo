How to use the visualization

This second milestone gives a basic overview of the statistics of the olympic games. The visualization is a world map that shows the number of olympic medals won by each country. The visualization is interactive and allows the user to see the number of medals won by each country in each olympic game. The visualization also allows the user to switch between the summer and winter olympic games. In order to switch between the two seasons, use the slider at the top right of the page.
  • Hover over a country to see its name
  • Click on a country to see the number of medals it has won in each Olympic games, as well as the most decorated athlete
